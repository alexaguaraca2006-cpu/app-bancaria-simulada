# app-bancaria-simulada
Aplicación bancaria simulada inspirada en el Banco Pichincha
## Implementación

El proyecto fue implementado en Python, desarrollando un módulo funcional
denominado `calculadora.py`, el cual permite realizar operaciones básicas
como suma, resta, multiplicación y división.

Se aplicó la metodología de Desarrollo Guiado por Pruebas (TDD), escribiendo
primero los casos de prueba unitarios y posteriormente implementando el
código hasta que todas las pruebas fueron aprobadas.

El control de versiones se realizó mediante GitHub, manteniendo un historial
de commits que evidencia el desarrollo incremental del proyecto.
